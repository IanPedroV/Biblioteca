Bem Vindo!

Esse projeto (que agora foi disponibilizado Open-Source) foi feito por um grupo de estudantes do Curso Técnico de Informática no IFMA - Campus Bacabal. O mesmo foi o trabalho final do curso. A função do projeto é de gerenciar uma biblioteca pública. Alugueis de Livro, cadastro de usuários, editoras, autores e funções do gênero estão disponíveis nesse programa escrito em Java que ainda conta com um interface gráfica para facilitar o uso. Componentes usados no trabalho: Java - Linguagem de Programação JDBC - Recurso para realizar queries SQL que já vem embarcado no Java Swing - Recurso gráfico para construção de aplicações para Desktop em Java
